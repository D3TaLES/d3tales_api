Processors

This module contains data parsers to process both computational and experimental raw data files. The D3TaLES parser then converts the processed data to the D3TaLES schema. It also contains a submodule for converting data from the backend D3TaLES database to frontend data and pushing the new data to the frontend database.

Full documentation can be found here.

Current Parsers:

  • Molecular DFT

    • Gaussian logfiles

    • Psi4 logfiles (in development)

  • Cyclic Voltammetry

    • Chi CV files

  • UV/Vis Spectroscopy

    • Excel output from ___ instrument
